# Contact Page with Footer

This project is a responsive *Contact Page* with a *Footer* for Panaversity, a cloud-native learning platform for AI, robotics, and next-gen skills. The page allows users to get in touch, choose a category (Teachers, Parents, NGOs, Volunteers), and includes a modern footer with company info and social links.

---

## Features

- Responsive *Navbar* with logo and navigation links.
- *Contact Form*:
  - Name, Email, Subject, Message.
  - Category selection: Teachers, Parents, NGOs, Volunteers.
  - Cloudflare Turnstile success notification.
- Modern *Footer* with:
  - Company description.
  - Social media icons (Facebook, Instagram, YouTube, LinkedIn).
  - Links to Company and Support pages.
  - WhatsApp community button.
- Fully responsive design for desktop, tablet, and mobile.
- Clean UI with *CSS Grid* and *Flexbox*.

---

## Technologies Used

- HTML5
- CSS3
- [Font Awesome](https://fontawesome.com/) for icons
- Google Fonts (Inter)
- Optional: Cloudflare Turnstile integration
